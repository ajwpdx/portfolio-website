About Me

Hello! I am Alex. I am a software developer from Portland, OR. I love the creative process, solving complex problems, and collaborating with others.

Prior to learning to code, I worked in the investment consulting industry. I am a big fan of spreadsheets and quantitative analysis, but a desire to use my creative capacities has led me to pursue a career in web development.

I have received an endorsement from Lambda School as a Full Stack Web Developer. My primary tech stack includes React / Redux on the frontend and Java / Spring on the backend, but I am always learning new technologies. I am currently building personal projects. Check out my the Projects tab above to see some of the projects that I have worked on or my GitHub to see my works in progress.

My professional journey wouldn’t be possible without the encouragement and help of others. If I can assist you with advice or mentoring, or if you would like to inquire about collaborating on a project, please don’t hesitate to reach out.

Citrics

An application used to compare statistics of major US cities. Users are able to search hundreds of major US cities and filter down by relevant statistics such as population and cost of living.

Citrics was a Lambda School Labs project. I worked on a team of multiple cross-functional developers. We redesigned an existing project with the goal of improving data visualizations and user experience.

PostHere

A dashboard interface where authorized users can create, save, edit, and delete a reddit post before posting it on Reddit. It integrates an API designed by our data science team which predicts which subreddit would be the best fit for the post.

PostHere was a Lambda School buildweek project. I was a lead front end developer on this project. We used React, Redux, Sass to build the app.

Wet My Plants

An application that enables a user to keep track of the watering frequency of their plants. Users can add, delete, and edit plants in their "Plant Family".

This was a Lambda School buildweek project and my first ever project as a front end developer! React and Sass were the primary tools used to build the front end functionality and styling.
